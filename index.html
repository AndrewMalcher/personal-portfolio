Sobre Mim

Sou um programador web apaixonado por tecnologia e inovação, com vasta experiência em HTML, CSS e JavaScript. Minha expertise inclui a utilização de frameworks modernos como React e pré-processadores CSS como Sass, que me permitem criar interfaces de usuário eficientes e visualmente atraentes. Atualmente, estou focado em projetos com Next.js, onde aplico as melhores práticas de desenvolvimento para construir aplicações web rápidas e escaláveis. Sempre buscando me atualizar com as últimas tendências do mercado, estou comprometido em entregar soluções de alta qualidade que superem as expectativas.

Projetos

Página de Pastelaria

Desenvolvi um site moderno e interativo para a Pastelaria Delícia utilizando HTML, CSS e JavaScript. O design atrativo e responsivo proporciona uma experiência envolvente para os clientes, destacando o cardápio variado e os sabores únicos da pastelaria. Recursos como animações sutis e um sistema de pedidos online intuitivo tornam o site não apenas funcional, mas também visualmente encantador.
